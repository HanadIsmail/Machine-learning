# CodeBasics ML Course Guide

Welcome! This folder contains the machine learning course materials I am studying step by step. Each topic has its own folder with exercises, resources, and projects to practice.

## How to use this guide

- Start with `1 python/` if you need a Python refresher.
- Move through the topics in order: NumPy, pandas, visualization, statistics, regression, and classification.
- Open the folders to find practice notebooks, worksheets, and examples.
- Read the folder names and comments in code files for quick guidance.

## Course folders

- `1 python/`
  - Basic Python practice and exercises.
  - Great for refreshing variables, loops, functions, and core programming skills.

- `2 numpy/`
  - NumPy array operations and matrix practice.
  - Useful for learning how numerical data is stored and processed in machine learning.

- `3 pandas/`
  - pandas data analysis workbooks and exercises.
  - Includes data merge/concat, group-by, and missing value handling.

- `4 Data Vizualization (matplotlib and seaborn)/`
  - Charts and plots using matplotlib and seaborn.
  - Helps visualize data, distributions, and model results.

- `5 Math and statistics/`
  - Statistics concepts used in ML: correlation, outliers, variance, standard deviation, and more.
  - Includes theory and hands-on exercises.

- `6 Regression/`
  - Regression topics like linear regression, multiple regression, regularization, and model evaluation.
  - Includes practical exercises and explained examples.

- `7 Supervised ML (Classification)/`
  - Classification algorithms and evaluation methods.
  - Includes logistic regression, precision/recall, confusion matrices, multiclass learning, and SVM.

## Notes for a friendly study flow

- Read the folder label first, then open the example files inside.
- Try running the code in small pieces to understand what each part does.
- Keep this guide open as a roadmap for your learning.

Happy studying!
